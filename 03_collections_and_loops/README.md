# Section 3: Collections, loops, and methods

This section covers Python’s **collection types** (lists, tuples, sets, dictionaries), how to **iterate** with **`for`** (including **`range`**, **`enumerate`**, and dict **`.items()`**) and **`while`**, how to use **`break`**, **`continue`**, and **`pass`** inside loops, and **common methods** on strings and collections.

**Indexing and slicing** for lists (and similar sequences) are in the **Lists** lesson. **Section 2** in this course focuses on **operators and conditions** (`if` / `elif` / `else`); **loops are introduced here** in Section 3, in the context of real data.

## Learning goals

By the end of this section, you will be able to:

- Create and modify **lists** (ordered, mutable sequences).
- Use **tuples** for fixed ordered groups (immutable).
- Use **sets** for unique unordered items and fast membership tests.
- Use **dictionaries** for key–value data.
- Index and slice **lists** (and other sequences) safely, including negative indices and steps.
- Write **`for`** loops over iterables, use **`range`**, and apply **`enumerate`** and dict **`.items()`** / **`.values()`** where appropriate.
- Write **`while`** loops and use **`break`**, **`continue`**, and **`pass`** where appropriate.
- Call typical **methods** (`append`, `get`, `add`, `split`, etc.) and know where to look up more.
