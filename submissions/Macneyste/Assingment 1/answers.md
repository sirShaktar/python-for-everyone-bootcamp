## Project Overview

This project is a simple text-based Python quiz. The program asks the user for their name, gives five Python questions, checks each answer, tracks the score, and prints the final result.

## Questions And Accepted Answers

1. **What keyword is used to define a function in Python?**
   - Accepted answer: `def`

2. **What symbol is used for addition in Python?**
   - Accepted answer: `+`

3. **What data type is used to store text in Python?**
   - Accepted answers: `str` or `string`

4. **Which keyword is used to repeat code in a loop?**
   - Accepted answers: `for` or `while`

5. **What function is used to get the length of a list or string?**
   - Accepted answers: `len` or `len()`

## Requirements Met

- The program uses `input()` to collect the user's name and answers.
- The program uses `print()` to show questions, feedback, and the final score.
- The program uses `if`, `elif`, and `else` statements.
- The program keeps score with an integer variable named `score`.
- The program prints the user's final score out of five questions.

## How To Run

```bash
python3 quiz.py
```
